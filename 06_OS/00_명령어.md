- pwd(print working directory)

	```shell
	$ pwd
	/home/ubuntu
	# 현재 작업중인 directory 정보 출력
	```
	
- cd(change directory)

  ```shell
  $ cd /home/ubuntu/myfolder
  $ cd ..
  # 경로 이동
  ```

- ls(list)

  ```shell
  $ ls
  # 현재 경로의 하위 디렉토리 및 파일 이름 표시
  
  $ ls -a
  # 숨김파일 및 디렉토리 ../ ./ 이름 까지 모두 표시
  
  $ ls -l
  # 파일에 대한 정보도 같이 표시
  
  $ ls -r
  # 역순으로 표시
  
  $ ls -t
  # 최종 수정 시간을 기준으로 표시
  ```

  ```shell
  # 조합가능 ex)
  
  $ ls -al
  # 숨김파일 및 디렉토리를 정보와 함께 표시
  
  $ ls -alrt
  # 숨김파일 및 디렉토리를 최종 수정 시간을 기준으로 역순으로 정보와 함께 표시
  ```
